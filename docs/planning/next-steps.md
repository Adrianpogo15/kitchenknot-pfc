# Proximos pasos recomendados

1. Inicializar `client/` con Expo Router o React Navigation.
2. Inicializar `server/` con Express y estructura modular.
3. Configurar PostgreSQL en Supabase.
4. Crear endpoints de autenticacion:
   - registro
   - login
   - perfil
   - baja de usuario
5. Definir la primera version de pantallas:
   - inicio
   - login y registro
   - listado y detalle de receta
   - perfil
   - listas de la compra

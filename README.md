# PFC - Aplicacion Movil de Recetas

Proyecto base para el desarrollo del PFC de DAM/DAM-like centrado en una aplicacion movil de recetas de cocina multiplataforma.

## Stack definido

- Frontend: React Native con Expo
- Backend: Node.js con Express
- Base de datos: PostgreSQL (recomendado para desplegar en Supabase)
- Compilacion APK: Expo EAS Build
- API externa de recetas: TheMealDB

## Estructura principal

- `client/`: aplicacion movil
- `server/`: API REST y logica de negocio
- `docs/`: documentacion funcional y tecnica

## Siguientes pasos

1. Inicializar el proyecto Expo dentro de `client/`.
2. Inicializar la API Express dentro de `server/`.
3. Crear la base de datos PostgreSQL con el esquema inicial.
4. Definir la primera pantalla de la aplicacion y su flujo.
